## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional untuk development)

### Installation

1. **Clone repository**
```bash
git clone https://github.com/username/elisabeth-portfolio.git
cd elisabeth-portfolio
```

2. **Add profile photo**
```bash
mkdir media
# Add your profile.jpg to media/ folder
```

```bash
# Option 1: Simple HTTP server
python -m http.server 8000

# Option 2: Node.js http-server
npx http-server

# Option 3: Live Server (VS Code Extension)
# Right-click index.html -> Open with Live Server
```

4. **Open browser**
```
http://localhost:8000
```

## üé® Customization

### Changing Colors
Edit CSS custom properties di bagian `:root`:

```css
:root {
  --primary-blue: #3b82f6;
  --secondary-blue: #60a5fa;
  --background-dark: #0a0a0a;
  --text-primary: #ffffff;
}
```

### Adding New Sections
1. Add HTML structure
2. Update navigation links
3. Add corresponding CSS styles
4. Update JavaScript scroll observer

### Modifying Animations
Edit keyframes di CSS:

```css
@keyframes customAnimation {
  0% { transform: translateY(0); }
  100% { transform: translateY(-20px); }
}
```
